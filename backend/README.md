# Backend Dailyhub (WIP)

Português Brasil:

- Servidor feito em NestJS.
- PostgreSQL como banco de dados.
- Docker para subir as aplicações, configuradas pelo Docker-Compose.
- GraphQL para a API do servidor.

English:

- Server made in NestJS.
- PostgreSQL as database.
- Docker-compose to run the applications.
- GraphQL API for the server.
